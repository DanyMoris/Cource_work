## ✨ Возможности

- **Загрузка из файла**: Импорт списка самолётов из текстового файла (формат таблицы с разделителем `|`).
- **Просмотр БД**: Вывод всей таблицы с колонками: №, Тип, Год, Мест, Груз(т), Инструктор.
- **Добавление / Удаление / Редактирование** записей.
- **Три вида сортировки основной базы**:
  - По грузоподъёмности (числовое поле).
  - По типу самолёта (строковое поле).
  - Структурная (по типу + фамилия инструктора).
- **Поиск**:
  - По одному признаку (фамилия инструктора или её часть).
  - По двум признакам (фамилия + тип).
- **Работа с перечнем (результатами поиска)**:
  - Просмотр перечня.
  - Сортировка перечня по году выпуска.
  - Сохранение перечня в отдельный файл‑отчёт.
- **Сохранение всей базы** в файл с сохранением форматирования.
- **Локализация**: Полностью русскоязычный интерфейс (используется кодировка Windows‑1251).

## 📄 Формат входного файла

Файл должен содержать строки следующего вида, разделённые символом `|`:
Тип самолёта|Год выпуска|Количество кресел|Грузоподъёмность(Т)|Фамилия инструктора


**Пример (`planes.txt`):**
Boeing 737-800|2015|189|23.5|Иванов
Airbus A320|2018|150|16.6|Петров
МС-21|2022|211|20.3|Сидоров
Superjet 100|2019|87|12.2|Кузнецов


Программа автоматически пропускает пустые строки и строки заголовка (содержащие `---` или `Тип`).

## 🔧 Сборка и запуск

### Требования
- Компилятор C++ с поддержкой стандарта **C++11** или новее.
- ОС **Windows** (программа использует `<windows.h>` для установки кодовой страницы консоли).

### Компиляция (пример для MinGW/g++)

```bash
g++ -o fleet.exe main.cpp airplane.cpp -std=c++11
Или через Visual Studio — создайте консольное приложение и добавьте все .cpp и .h файлы в проект.

Запуск

fleet.exe

🧱 Описание классов
Airplane
Модель данных одного воздушного судна.

Поля: type, year, seats, payload, instructor.

Методы: геттеры/сеттеры, printRow() для вывода на экран, saveRow() для сохранения в файл.

FleetManager
Управляющий класс, содержит всю бизнес-логику.

Хранилище: std::vector<Airplane> fleet — основная база; searchList — результаты последнего поиска.

Основные публичные методы:

loadFromFile() / saveToFile()

addRecord(), deleteRecord(), editRecord()

sortByPayload(), sortByType(), sortStructural()

search(), showSearchList(), sortSearchList(), saveSearchList()

Вспомогательные: форматированный вывод таблиц (printHeader, printSeparator, printAnyList).
